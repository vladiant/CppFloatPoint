# Floating point arithmetic

## Special values

### Infinity
* `inf + x = inf` when `x > -inf`
* `inf - inf = NaN`
* `inf * 0 = NaN`
* `0 / 0 = NaN`
* `inf / inf = NaN`

### NaN
* Result of every arithmetic with NaN is NaN

## Exceptions

* Overflow
* Underflow
* Invalid
* Inexact
